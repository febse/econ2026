# Manim starter: population sampling

This folder contains a base Manim scene that:
- creates a synthetic population of stylized people,
- assigns each person two attributes (`height_cm`, `weight_kg`),
- animates drawing a random sample from that population.

## File
- `population_sampling.py`

## Run
From the project root:

```bash
uv run manim -pql anime/population_sampling.py PopulationSampling
```

Use `uv run manim -pqh anime/population_sampling.py PopulationSampling` for higher quality output.
